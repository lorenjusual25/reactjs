## Presentacion
Aplicacion e-commerce de tecnologia desarrollada en React y Vite para el curso de React, que permite a los usuarios navegar en la pagina principal, entre categorias y ver mas detalles de cada producto. Tambien poder agregarlos al carrito y realizar compras con su correspondiente ID de orden.

## Link en Vercel
- https://reactjs-sable-five.vercel.app/

## Caracteristicas:
- Navegacion por categorias
- Ver en detalle cada producto y sus reviews
- Carrito de compras interactivo
- Sistema de ordenes con Firebase
- Notificaciones con react hot toast

## Tecnologias:
- **react**
- **vite**  
- **firebase** - Base de datos en la nube
- **react-hot-toast** - Sistema de notificaciones
- **react-router-dom** - Sistema de rutas
- **DummyJSON API** - Fuente de datos de productos
